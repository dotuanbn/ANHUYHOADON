## ⚡ LỆNH HAY DÙNG NHẤT

### 1. Update Web (Dùng nhất!)
```bash
git add . && git commit -m "update" && git push
```
✅ Chờ 30 giây → Web update!

### 2. Hoặc dùng script
```bash
deploy-update.bat    # Double-click
# Chọn [2]
```

### 3. Test Local
```bash
npm run build
npm run preview
```
